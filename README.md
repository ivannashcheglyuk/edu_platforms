# WEB103 Prework - *Educational Platforms*

Submitted by: **Ivanna Shcheglyuk**

About this web app: **Educational Platforms is a React app that curates the best online learning resources. Users can explore, add, edit, and delete their favorite educational platforms, each featuring a name, a direct link, a description of their content, and an optional image. The app makes it easy to discover and organize valuable learning resources in a visually engaging, cinematic card layout.**

Time spent: **12** hours
